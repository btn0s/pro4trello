Pro4Trello hits version 3.1

A lot of issues fixed, some feature wishes granted to great users!

1. WHAT'S NEW?

1.1 AUTO-HIDE FOOTER

You can now set the footer to automatically hide. Could not be easier really...

Just hit the menu and check that "Auto-hide footer" checkbox.

1.2 NEW STUFF IN MENU

Easy access to global settings (applicable accross all boards) and helper videos.

1.3 HIDE ALL LISTS

Just like you can show all lists, you can now simply hide all lists from the "Lists" menu. Ez.

2. WHAT WAS FIXED?

Thank you for your patience!

All languages are now supported for list/card titles => filters and list options work

Custom backround is now showing (following Trello updates)

Pretty card descriptions are now working again

Prioritized cards are now rendering with proper color

FAQ

We still have to wait really long for updates, why?

Life formula is now: family + hard business + zero help = absolutely no time. It is what it is. You are still awesome with feedback and ideas so keep it comming, progress is being made!

What happened with my feature request?

Thank you for suggesting new features and (more often) pointing out Pro4Trello's flaws. Please stay patient until the time comes for your feature request to be a part of one of the next versions. And rest assured it will one day.

How can I contribute?

1. Review the extension

If you think Pro4Trello is awesome - you can leave a rating in the Chrome Store. This will help other people learn about the extension. Also make sure to leave your feedback or a feature request!

Do you have a blog, a video channel or write for tech. media? Consider reviewing Pro for Trello in a video or an article. The author is always available for a comment or any kind of feedback.

2. Make a pull request

If you can help you likely should, Pro4Trello is public on GitHub: github/mdjekic/pro4trello

3. Donate

If Pro for Trello makes your work or life easier please consider donating.
